Questions, answered.

Is it really free?

Yes. The app is free to download and use, and there's no account, no ads, and no "pro" version. If it saves your Saturday, you can buy the developer a coffee, but that's entirely optional.

Can it break my mods or saves?

Disabling a mod only renames the file (adding a .disabled suffix) so the game skips it; nothing is ever deleted, and re-enabling restores the exact original file. Your saves are never touched.

Windows says the app is unrecognized. Is it safe?

The app isn't code-signed yet (certificates are expensive for a free project), so SmartScreen shows a warning on first run. Choose More info → Run anyway. The full source code is public on GitHub, and every download is built automatically from that code by GitHub's own servers.

Does it work with the Ultimate Collection / localized games?

Yes. It detects localized user folders ("Los Sims 3", "Die Sims 2"…), the Sims 2 Ultimate Collection, Documents folders moved by OneDrive, and multiple installs of the same game. If all else fails, you can point it at any folder manually.

Where do I report a bug or ask for a feature?

On the GitHub issues page; bug reports and ideas are very welcome. There's also a wiki with a user guide and FAQ.
